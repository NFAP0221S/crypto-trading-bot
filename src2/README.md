# Crypto AI Trading Bot

Python과 OpenAI GPT-4를 활용한 암호화폐 자동 거래 봇입니다.

## 주요 기능

- 업비트 API를 통한 시장 데이터 수집
- 기술 지표 계산 (SMA, EMA, RSI, MACD, Bollinger Bands 등)
- 최신 뉴스 데이터 수집
- 공포와 탐욕 지수 수집
- 차트 이미지 캡처
- GPT-4를 통한 데이터 분석 및 매매 결정
- 자동 매매 실행
- 거래 내역 데이터베이스 저장
- 로깅 및 예외 처리
- 스케줄러를 통한 자동화
